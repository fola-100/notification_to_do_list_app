# # AI Character To-Do Reminder App

## Overview
This project is a character-driven to-do reminder application where users can create tasks, goals, habits, 
and reminders while selecting a fictional character personality.Instead of generic reminder messages, the
application responds in the style of the selected character.

## Current Features
- Task reminders
- Character-based responses
- Notification system
- Saved tasks
- Future AI-generated dialogue

## Future Goals
- Desktop notifications
- AI-generated character responses
- GUI interface
- Database integration
- Voice interaction

## Technologies Used
- Python
- JSON/file storage
- schedule library
- notification library
